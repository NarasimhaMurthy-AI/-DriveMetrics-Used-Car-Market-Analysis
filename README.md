# DriveMetrics — Used Car Market Analysis

**One-page interactive Power BI dashboard** analyzing used car listings: price, kilometers, fuel type, transmission, engine capacity and more.  
Built using: **Excel**, **SQL Server (local)**, **Power BI Desktop**, with DAX measures for dynamic KPIs.

## Project summary (short)
This project takes raw used-car CSV files, consolidates them into a single SQL table (`all_cars`), performs cleaning and shaping, and builds an interactive Power BI report with KPI cards, charts, and slicers. The dashboard is designed for business stakeholders to get instant market signals and drill into segments.
